# 特殊票券二维码功能说明

## 功能概述

在B端特殊票券管理页面，管理员可以查看每张票券的详情，并生成对应的二维码。

## 已完成的修改

### 1. 前端功能增强

**文件：`frontend-b/src/views/ticket/SpecialTicketList.vue`**

新增功能：
- 在操作列添加"查看详情"按钮
- 点击后弹出详情对话框，显示：
  - 票券完整信息（编号、状态、创建时间、核销时间、核销人等）
  - 根据票券编号自动生成的二维码
  - 下载二维码按钮

### 2. 依赖库添加

**文件：`frontend-b/package.json`**

新增依赖：
- `qrcode`: ^1.5.3 - 用于生成二维码
- `@types/qrcode`: ^1.5.5 - TypeScript类型定义

## 使用说明

### 查看票券详情和二维码

1. 登录B端管理系统
2. 进入"门票管理" -> "特殊票券管理"
3. 在票券列表中，点击任意票券的"查看详情"按钮
4. 弹出详情对话框，显示：
   - 票券基本信息
   - 二维码（300x300像素）
5. 可以点击"下载二维码"按钮，保存为PNG图片

### 二维码内容

- 二维码内容：票券编号（如：ST00000001）
- 尺寸：300x300像素
- 格式：PNG
- 颜色：黑白

### 核销流程

1. 用户持有票券二维码（打印或电子版）
2. 到展馆入口出示二维码
3. 工作人员使用C端核销系统扫描
4. 系统识别票券编号（ST开头）
5. 自动调用特殊票券核销接口
6. 核销成功，票券状态变为"已使用"

## 技术实现

### 二维码生成

使用 `qrcode` 库的 `toCanvas` 方法：

```typescript
await QRCode.toCanvas(canvas, ticketCode, {
  width: 300,        // 二维码宽度
  margin: 2,         // 边距
  color: {
    dark: '#000000', // 前景色（黑色）
    light: '#FFFFFF' // 背景色（白色）
  }
})
```

### 下载功能

将Canvas转换为PNG图片并下载：

```typescript
const url = canvas.toDataURL('image/png')
const link = document.createElement('a')
link.href = url
link.download = `票券_${ticketCode}.png`
link.click()
```

## 注意事项

1. **二维码内容**：只包含票券编号，不包含其他信息
2. **安全性**：票券编号是唯一的，核销时会验证状态
3. **打印建议**：建议打印尺寸至少3x3厘米，确保扫描清晰
4. **批量打印**：可以使用"导出票券"功能导出所有编号，然后批量生成二维码

## 批量生成二维码（可选）

如果需要批量生成所有票券的二维码，可以使用以下Python脚本：

```python
import qrcode
import csv

# 读取导出的CSV文件
with open('特殊票券.csv', 'r', encoding='utf-8') as f:
    reader = csv.reader(f)
    next(reader)  # 跳过表头
    
    for row in reader:
        ticket_code = row[0]
        
        # 生成二维码
        qr = qrcode.QRCode(
            version=1,
            error_correction=qrcode.constants.ERROR_CORRECT_L,
            box_size=10,
            border=2,
        )
        qr.add_data(ticket_code)
        qr.make(fit=True)
        
        img = qr.make_image(fill_color="black", back_color="white")
        img.save(f'qrcodes/{ticket_code}.png')
        
print("批量生成完成！")
```
