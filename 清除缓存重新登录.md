# 清除缓存重新登录

## 问题
修改了token处理方式后，浏览器localStorage中还保存着旧的登录状态，需要清除。

## 解决方法

### 方法1：浏览器开发者工具清除
1. 打开 http://localhost:5174
2. 按F12打开开发者工具
3. 切换到"Application"标签（或"应用程序"）
4. 左侧找到"Local Storage" → "http://localhost:5174"
5. 删除`seller_token`项
6. 刷新页面
7. 重新登录

### 方法2：浏览器无痕模式
1. 打开浏览器无痕/隐私模式
2. 访问 http://localhost:5174
3. 登录测试

### 方法3：清除所有站点数据
1. 浏览器设置 → 隐私和安全
2. 清除浏览数据
3. 选择"Cookie和其他网站数据"
4. 清除数据

## 验证
登录后，在开发者工具Console中执行：
```javascript
console.log(localStorage.getItem('seller_token'))
```

应该看到一个JWT token（不带Bearer前缀）。

## 测试步骤
1. 清除缓存
2. 访问 http://localhost:5174
3. 登录：seller / 123456
4. 点击"单号核销"
5. 输入：T1734240000000TEST1
6. 点击"核销"
7. 应该成功！
