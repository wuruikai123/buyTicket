# 关于展厅功能说明

## 功能概述
实现了A端"关于展厅"页面和B端管理功能，支持自定义标题、正文、背景图和介绍图。

## 完成的工作

### 1. 数据库
✅ 创建 `about_config` 表
- 字段：id, title, content, background_image, intro_image1, intro_image2
- 标题限制：不超过10个字
- 包含默认数据

### 2. 后端接口
✅ 创建实体类：`AboutConfig.java`
✅ 创建Mapper：`AboutConfigMapper.java`
✅ 创建Service：`AboutConfigService.java`
✅ A端接口：`GET /api/v1/about/config` - 获取配置
✅ B端接口：
- `GET /api/v1/admin/about/config` - 获取配置
- `POST /api/v1/admin/about/config` - 更新配置

### 3. A端页面
✅ 创建 `About.vue` 页面
- 背景图片层（最底层）
- 深色渐变层（顶部透明渐变到 #2c201d）
- 内容层（白色文字 #ffffff）
- 返回按钮
- 标题显示
- 正文显示
- 两张介绍图片

✅ 添加路由：`/about`
✅ 首页右上角菜单图标点击跳转到关于展厅

### 4. B端管理页面
✅ 创建 `AboutManagement.vue` 页面
- 标题输入（限制10字）
- 正文输入（多行文本框）
- 背景图片URL输入+预览
- 介绍图1 URL输入+预览
- 介绍图2 URL输入+预览
- 保存按钮
- 预览按钮（新窗口打开A端页面）

✅ 添加到侧边栏："系统设置" → "关于展厅"
✅ 添加路由：`/system/about`

## 页面设计

### A端样式
- **背景层**：背景图片，cover模式
- **渐变层**：从顶部透明渐变到底部 #2c201d
- **文字颜色**：#ffffff（白色）
- **标题**：32px，居中，加粗
- **正文**：16px，行高1.8，两端对齐
- **介绍图**：圆角12px，阴影效果

### B端管理
- 表单布局，最大宽度800px
- 图片URL输入+实时预览
- 标题字数限制提示
- 保存和预览按钮

## 使用流程

### B端配置
1. 登录B端管理系统
2. 进入"系统设置" → "关于展厅"
3. 填写标题（不超过10字）
4. 填写正文内容
5. 输入背景图片URL（建议16:9）
6. 输入介绍图1和介绍图2的URL
7. 点击"保存配置"
8. 可点击"预览效果"查看A端显示

### A端查看
1. 打开A端首页
2. 点击右上角菜单图标（三横线）
3. 进入"关于展厅"页面
4. 查看配置的内容
5. 点击返回按钮返回首页

## 注意事项

1. **图片URL**：建议使用完整的HTTP/HTTPS URL
2. **背景图比例**：建议使用16:9的图片作为背景
3. **标题限制**：前后端都有10字限制验证
4. **默认数据**：首次安装会插入默认配置
5. **单例模式**：系统只维护一条配置记录
