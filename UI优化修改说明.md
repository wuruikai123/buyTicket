# UI 优化修改说明

## 修改内容总结

根据产品需求，对A端（用户端）进行了以下UI优化：

### 1. 底部导航栏（BasicLayout.vue）

✅ **删除"文创商城"导航标签**
- 原因：文创商城功能非本期开发内容
- 修改：注释掉底部导航栏中的"文创商城"项
- 效果：底部导航只显示"首页"、"近期展览"、"我的"三个标签
- 代码位置：`frontend-a/src/layouts/BasicLayout.vue` 第 32-35 行
- 注意：商城页面和路由代码已保留，用户仍可通过直接访问 `/mall` 路径进入

### 2. 个人中心页面（Profile.vue）

#### 隐藏的功能（保留代码，仅隐藏显示）

✅ **隐藏"我的商城订单"标签页**
- 原因：文创商城功能非本期开发内容
- 修改：注释掉商城订单标签页的显示
- 代码位置：`frontend-a/src/views/Profile.vue` 第 48-54 行

✅ **隐藏"排序"和"筛选"功能**
- 原因：排序和筛选功能非本期开发内容
- 修改：注释掉整个 filter-bar 区域
- 代码位置：`frontend-a/src/views/Profile.vue` 第 57-68 行

✅ **隐藏"地址簿"按钮**
- 原因：地址簿功能非本期开发内容
- 修改：注释掉地址簿按钮，只保留"修改信息"按钮
- 代码位置：`frontend-a/src/views/Profile.vue` 第 23-27 行

### 3. 近期展览页面（Exhibitions.vue）

✅ **保留"进行中"和"待开始"标签页**
- 效果：用户可以切换查看不同状态的展览
- 代码位置：`frontend-a/src/views/Exhibitions.vue` 第 3-18 行

✅ **隐藏"排序"和"开展时间最近优先"文字**
- 原因：排序功能非本期开发内容
- 修改：注释掉排序/筛选栏
- 效果：展览列表直接显示，无筛选栏
- 代码位置：`frontend-a/src/views/Exhibitions.vue` 第 20-27 行

### 4. 首页轮播图（Home.vue）

✅ **添加左右翻页按钮**
- 修改：设置 `arrow="hover"`，鼠标悬停时显示翻页按钮
- 效果：轮播图自动播放，鼠标悬停时显示左右箭头
- 样式：半透明白色圆形按钮，带毛玻璃效果
- 代码位置：`frontend-a/src/views/Home.vue` 第 14 行

### 5. 关于展厅页面

⚠️ **"关于展厅"页面点不开**
- 原因：需要在管理端（B端）配置展厅信息
- 解决方案：
  1. 在B端管理后台添加"展厅配置"功能
  2. 配置内容包括：
     - 背景图片（16:9比例）
     - 两张介绍图（比例不限）
     - 标题（不超过10个字）
     - 纯文字介绍
  3. 样式要求：
     - 正文和标题为白色 (#ffffff)
     - 深色背景色 (#2c201d)
     - 顶部渐变至透明
     - 背景图片叠在深色背景的渐变下层

## 后续开发建议

### 关于展厅页面开发

需要在B端（管理端）添加以下功能：

1. **展厅配置管理**
   - 路由：`/admin/venue-config`
   - 功能：
     - 上传背景图片（16:9）
     - 上传介绍图片（2张，比例不限）
     - 输入标题（限制10字）
     - 输入介绍文字（富文本编辑器）
     - 预览功能
     - 保存/发布功能

2. **A端展厅页面**
   - 路由：`/about`
   - 功能：
     - 从后端API获取展厅配置
     - 按照设计稿渲染页面
     - 响应式适配

3. **后端API**
   - 接口：`GET /api/v1/venue/config`
   - 接口：`POST /api/v1/admin/venue/config`（管理端）
   - 数据库表：`venue_config`
