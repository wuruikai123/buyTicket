# 特殊票券简化版功能说明

## 功能概述

简化了特殊票券管理，B端只保留生成按钮，点击后直接生成票券并导出所有二维码为ZIP压缩包。C端核销特殊票券时显示"特邀VIP"。

## B端功能

### 特殊票券生成页面

**位置：** 门票管理 -> 特殊票券管理

**功能：**
1. 输入生成数量（1-10000张）
2. 点击"生成并下载二维码"按钮
3. 系统自动：
   - 调用后端生成票券
   - 为每张票券生成二维码（300x300 PNG）
   - 将所有二维码打包成ZIP文件
   - 自动下载ZIP文件

**文件命名：**
- ZIP文件：`特殊票券二维码_时间戳.zip`
- 二维码文件：`ST00000001.png`, `ST00000002.png` ...

**操作流程：**
```
输入数量 → 点击生成 → 等待处理 → 自动下载ZIP
```

**进度提示：**
- 正在生成票券...
- 正在获取票券列表...
- 正在生成二维码 1/5000...
- 正在打包下载...

## C端功能

### 核销显示

**特殊票券核销成功显示：**
- 展览名称：特邀VIP
- 购买账号：特邀VIP
- 核销时间：当前时间

**普通订单核销成功显示：**
- 展览名称：门票订单
- 购买账号：用户
- 核销时间：当前时间

## 技术实现

### B端核心代码

```typescript
// 生成并下载
const handleGenerateAndDownload = async () => {
  // 1. 生成票券
  await specialTicketApi.batchGenerate(count)
  
  // 2. 获取票券列表
  const tickets = await specialTicketApi.export(0)
  
  // 3. 生成二维码并打包
  await generateQRCodesZip(tickets)
}

// 生成二维码ZIP
const generateQRCodesZip = async (tickets) => {
  const zip = new JSZip()
  
  for (let ticket of tickets) {
    // 生成二维码到canvas
    await QRCode.toCanvas(canvas, ticket.ticketCode, {
      width: 300,
      margin: 2
    })
    
    // 转换为blob
    const blob = await canvas.toBlob()
    
    // 添加到zip
    zip.file(`${ticket.ticketCode}.png`, blob)
  }
  
  // 下载zip
  const zipBlob = await zip.generateAsync({ type: 'blob' })
  download(zipBlob, '特殊票券二维码.zip')
}
```

## 使用流程

### 管理员操作

1. 登录B端管理系统
2. 进入"门票管理" -> "特殊票券管理"
3. 输入生成数量（如5000）
4. 点击"生成并下载二维码"
5. 等待处理（约1-2分钟）
6. 自动下载ZIP文件
7. 解压ZIP文件
8. 打印或分发二维码

### 用户使用

1. 获得特殊票券二维码
2. 到展馆入口出示
3. 工作人员使用C端扫码
4. 显示"特邀VIP 核销成功"
5. 允许入场

## 注意事项

1. **生成时间**：5000张票券约需1-2分钟
2. **文件大小**：每个二维码约10KB，5000张约50MB
3. **浏览器要求**：建议使用Chrome或Edge浏览器
4. **网络要求**：需要稳定的网络连接
5. **存储空间**：确保有足够的磁盘空间

## 性能优化

## 错误处理

### 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 生成超时 | 数量过多 | 减少生成数量 |
| 下载失败 | 浏览器拦截 | 允许下载 |
| ZIP损坏 | 生成中断 | 重新生成 |
| 二维码模糊 | 打印设置 | 调整打印质量 |

## 常见问题

**Q: 生成5000张需要多久？**
A: 约1-2分钟，取决于电脑性能和网络速度。

**Q: ZIP文件有多大？**
A: 5000张约50MB，每张二维码约10KB。

**Q: 可以重复生成吗？**
A: 可以，每次生成都是新的票券编号。

**Q: 二维码可以打印吗？**
A: 可以，建议打印尺寸至少3x3厘米。

**Q: 如何批量打印？**
A: 解压ZIP后，使用图片批量打印工具。

**Q: 核销后能看到记录吗？**
A: 可以，在数据库中查看special_ticket表。
